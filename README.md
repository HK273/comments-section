# Comments Section

## Tools & Technologies

This project uses the following tools & technologies

- [NextJS](https://nextjs.org/docs/app/getting-started)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/docs/installation) - For styling
- [shadcn/ui](https://ui.shadcn.com/) - For additional components
- [Drizzle ORM](https://orm.drizzle.team/docs/get-started/postgresql-new) - For interactions with the Neon Postgres instance
- [Neon](https://neon.com/docs/guides/nextjs) - For hosted Postgres instance
- [Better Auth](https://www.better-auth.com/docs/integrations/next) - For authentication
- [Jest](https://jestjs.io/) & [React Testing Library](https://testing-library.com/) - For testing
- [TSDoc](https://tsdoc.org/) - For TypeScript specific code annotations
- [Eslint](https://eslint.org/) & [Prettier](https://prettier.io/) - For formatting and linting
- [Husky](https://typicode.github.io/husky/) & [lint-staged](https://github.com/lint-staged/lint-staged) - For pre commit checks
- [Zustand](https://zustand-demo.pmnd.rs/) - State management for React
- [Date-fns](https://date-fns.org/) - Date utilities
